유튜브 허위·과장 광고 의심신호 탐지시스템 서비스를 만들거야.
배포는 구글 크롬 확장 프로그램으로 할 거야.
기능 중에 허위광고 탐지 시 보고서 웹사이트를 만들어서 링크를 제공해야하고, 복잡한 서버 로직은 필요없어서 Next.js 기반 풀스택으로 구현할 예정.

확장 프로그램은 plasmo 프레임워크로 만들어줘.

폴더 구조는 의존성이 꼬이지 않게 다음과 같이 만든다.

- `extension/`  크롬 확장 (plasmo)
- `next/`       보고서 웹 + 포스트-MVP 서버 (Next.js 16)

이 시스템은 광고의 합법/불법을 자동으로 최종 판정하는 기계가 아니다.대신, 영상 속 표현 중에서 법규상 문제가 될 가능성이 있는 문장과 신호를 자동으로 찾아서, 사람이 검토할 수 있도록 위험도와 근거를 제시하는 시스템이다.본질은 판사처럼 최종 판결하는 시스템이 아니라, 탐지기 + 근거 제시기 + 선별기라는 거다.

해결하려는 것
- 건강식품·다이어트 광고성 영상의 위법 표현 탐지
- 크롬 확장 프로그램으로 실시간 경고

해결하지 않는 것
- 제품의 진위 판별·의학적 사실 검증
- 광고 차단·삭제

시스템 파이프라인
1. 확장 프로그램이 유튜브 영상 감지
3. 확장 프로그램이 유튜브 자막을 추출하고 rule-based 알고리즘을 돌린다..
4. rule-based에서 걸리지지 않은 영상만 AI모델에게 전달해 허위 광고를 판별한다.
5. 판별 결과를 법 조항에 근거하여 보고서를 만들어 제공한다.
6. 서버는 이 보고서를 웹사이트에 게시하여 링크만 사용자에게 제공한다. 또한, 유튜브 영상 시청을 방해하지 않는 선에서 허위 광고 탐지 UI를 표시한다.
7. 링크를 클릭하면 웹사이트로 이동한다. 이때 새로운 탭을 생성해야 한다.


검증 단계
1. 1차에서 rule-based기반으로 영상을 걸러낸다.
2. 그 다음 AI 모델과 연결하여 실제 허위광고를 판별한다. 아직 LLM을 사용할지 파인튜닝 모델을 사용할지 결정하지 않아서 MVP에서 제외한다.


1차 목표: - rule-based 기반 탐지 알고리즘을 구현하여 1차 배포를 준비한다.


## 주의사항
이 내용을 바탕으로 사용자와 논의 후 docs/_template 폴더 기반으로 새로운 폴더와 파일을 세팅해줘.

1. AI 판별 모델 연결
2. 보고서 작성 LLM
